「mazedepth.py」が深さ優先探索、「mazewidth.py」が幅優先探索です。
実行するときは、コマンドライン引数で「testモード」と「runモード」を選択してください。
例）「python3 (ファイル名)　test」「python3 (ファイル名)　run」

testモード：10×10の迷路を探索し、ターミナル上に探索済みルートを出力します。
runモード：100×100の迷路を100個探索し、計算量とオープンリストの最大量を出力します。

迷路データについて：
test用はmap1010というファイルに、run用はmazedata内のmap0～map99というファイルにあります。
「0」が通路、「1」が壁、「#」が探索済みルートを表しています。

動作環境によってはファイルの読み込みが正常にされない可能性があります。
その場合、以下を参照してください

testモードのエラー：75行目のファイルパス「"Maze/map1010"」を環境に合わせて変更してください。
runモードのエラー：79行目のファイルパス「"Maze/mazedata"」を環境に合わせて変更してください。
　　　　　　　　　 なお、「/map"+str(i)」の部分はなるべく変更しないようにしてください。
